# Assignment 1 - Node.js JSON API

## How to run locally

1. Clone the repository
git clone https://github.com/Vigneswaran-Saravanan/assignment-1-node.js-web-app.git

2. Install dependencies
npm install

3. Create `.env` file (use the one uploaded on Blackboard)

4. Run the server
npm start

5. Open in browser
http://localhost:8888

6. Login using credentials:
- Username: AdminUser
- Password: adminpassword
